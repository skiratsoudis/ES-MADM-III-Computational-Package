# Computational Outputs

This folder contains the consolidated computational output workbook supporting the ES-MADM III manuscript results.

## File

```text
ES_MADM_III_Results_Computational_Outputs.xlsx
```

---

## Purpose

This workbook contains the final computational outputs generated by the ES-MADM III Decision Studio for the five-scenario case study.

It serves as the reference output file for verifying the manuscript tables, diagnostic interpretation, benchmark comparison, and graphical results. The workbook is included to support reproducibility and to allow reviewers and readers to compare independently generated outputs with the final results reported in the manuscript.

---

## Content

The output workbook contains the main result layers required to verify the manuscript analysis, including:

- representative subjective, objective, and integrated weights;
- operational entropy values;
- operational diversification degrees;
- Integrated Criteria Importance values;
- final alternative scores;
- final rankings;
- system-level entropy measures;
- mutual information values;
- diagnostic indices;
- scenario-relative NMGI synthesis;
- benchmark comparison outputs;
- scenario-level comparative summaries.

---

## Use for Reproducibility

To reproduce the workbook:

1. Open the ES-MADM III Decision Studio from the `app/` folder.
2. Import the S1 baseline input workbook from the `data/` folder.
3. Import S2, S3, S4A, and S4B through the scenario portfolio workflow.
4. Execute the ES-MADM III calculations.
5. Export the consolidated results workbook.
6. Compare the exported workbook with `ES_MADM_III_Results_Computational_Outputs.xlsx`.

The reproduced workbook should match the reference output when the same input files and computational settings are used.

---

## Relation to Manuscript Tables

The workbook supports the numerical content reported in the manuscript, including:

- scenario-level input and output summaries;
- representative integrated weights;
- Integrated Criteria Importance;
- final alternative scores and rankings;
- system-level entropy and information measures;
- diagnostic indices;
- benchmark comparison against fuzzy PROMETHEE II;

The file is therefore part of the reproducibility evidence for the reported case-study results.

## Notes

This file corresponds to the final computational results used in the manuscript. It should be treated as a reference output workbook, not as an input file.

Earlier output files, intermediate exports, and development-stage result workbooks are intentionally excluded to avoid ambiguity.
